# Plataforma SiteWhere

La plataforma SiteWhere 2.x presenta una arquitectura completamente rediseñada basada en un enfoque moderno de microservicios. Ha sido diseñado desde cero para ser confiable, de alto rendimiento, de baja latencia y escalabilidad dinámica utilizando tecnologías como Apache Kafka y Docker. La intención es continuar soportando la mayoría de modelo de datos y API de la plataforma 1.x, pero hacínedolo de una manera más modular la cual se adherirse a los flujos de trabajo de desarrollo e implementación modernos.
