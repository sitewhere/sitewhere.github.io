SiteWhere Community Edition is an open source IoT Application Enablement Platform. It provides a highly-configurable system that facilitates the ingestion, storage, processing, and integration of device data for one or more system tenants. SiteWhere also provides software development kits and agents for many of the most popular device technologies to allow them to interact and be controlled by the server.
