SiteWhere is an industrial-strength open source IoT Application Enablement Platform that facilitates the ingestion, storage, processing, and integration of device data at massive scale. SiteWhere is based on a modern microservices architecture and has been designed from the ground up for reliable, high throughput, low latency processing and dynamic scalability using technologies such as Apache Kafka and Docker.
